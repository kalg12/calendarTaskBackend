# Create Task Backend
## This is a backend repository for the task frontend project, developed by Kevin.

## Installation
### Clone the repository
```bash
git clone
```
### Install dependencies
```bash
npm install
```
### Run the project
```bash
npm run dev
```
### Run the project in production
```bash
npm run start
```
